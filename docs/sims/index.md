# MicroSims

This textbook includes 33 interactive MicroSims that bring blockchain and trust technology concepts to life through hands-on exploration.

## How to Use MicroSims

Each MicroSim is an interactive visualization embedded directly in chapter content. You can also access them individually from this list. MicroSims are designed to be explored — adjust sliders, click nodes, and hover over elements to learn.
